# little-jesus-vue

## Project setup

```
yarn install
```

<br>

### Compiles and hot-reloads for development

```
yarn run serve
```

<br>

### Compiles and minifies for production

```
yarn run build
```

<br>

### Run your tests

```
yarn run test
```

<br>

### Lints and fixes files

```
yarn run lint
```
